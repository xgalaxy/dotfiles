# dotfiles

Only works on OSX.
This is my configuration files for development, primarily in C/C++, C#, Java, and Javascript.

## installing
```
git clone https://github.com/xgalaxy/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
rake install
```

## updating
```
cd ~/.dotfiles
rake update
```

## uninstalling
```
cd ~/.dotfiles
rake uninstall
```
