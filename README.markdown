## installing
```
git clone https://github.com/xgalaxy/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
rake install
```

## updating
```
cd ~/.dotfiles
rake update
```

## uninstalling
```
cd ~/.dotfiles
rake uninstall
```
