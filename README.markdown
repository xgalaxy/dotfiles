# dotfiles

Only works on OSX.
This is my configuration files for development, primarily in C/C++, C#, Java, and Javascript.

## installing
```
git clone --recursive https://github.com/xgalaxy/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
rake install
```

## updating
```
cd ~/.dotfiles
rake update
```

## upgrading modules
```
cd ~/.dotfiles
rake upgrade
```

## uninstalling
```
cd ~/.dotfiles
rake uninstall
```
